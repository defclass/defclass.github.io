我的互联网生存工具

1. 软件

1.1. 文字

由于日常用电脑以写代码和文字为主，文字相关的工具会很常用。

1.1.1. 编辑器

  • Sublime: 写临时，简单的文本。
  • Emacs: 写日报。
  • Vim： 服务器上的常用工具 。
  • Typora：写 Markdown 文档。
  • overleaf: 写 Latex。

1.1.2. 笔记软件

  • TiddlyWiki on Node.js：将严肃的思考结论构建成卡片笔记系统。重度使用双链功能来链接不同的卡片。
    • Krystal
    • Relink
    • TiddlyWeb
  • Logseq：记不那重要的笔记，如流水账。用到双链功能，Youtube 链接等。
  • Anki: 记忆卡片。

1.1.3. IDE

  • Jetbrans All Products Pack
  • Github Copilot

1.1.4. 编程辅助

  • DevToys: 编程工具集。
  • Insomnia: 构造 HTTP 请求（类似于 Postman )。
  • Orbstack: 容器管理。
  • ColorSlurp: 拾色器。

1.2. 效率

  • Rectangle: 分屏工具。
  • XMind ZEN: 思维导图。
  • RIME: 五笔输入法、日语输入法。

1.3. 邮件

  • GMail
  • Thunderbird
  • Outlook Mail （已弃用）

1.4. 沟通

  • Telegram
  • Slack
  • Discord

1.5. 系统

  • Raycast: 快捷唤起。
  • VeraCrypt: 磁盘全盘加密。
  • OmniDiskSweeper： Mac 硬盘清理。
  • CleverFiles Cleaner: Mac 硬盘清理。 有时容量会算不准，后续逐渐弃用。
  • VitrualBox: 虚拟化工具。
  • Clocker: 时区工具。
  • Motrix: 下载工具。
  • Keka: 压缩工具。
  • Weather for Status：天气工具。
  • AppCleaner: 卸载工具。

1.6. 字典

  • Apple dictionary

1.7. 影音

  • Final Cut Pro: 视频编辑。
  • IINA
  • VLC
  • Foobar2000
  • LICEcap: 录屏。
  • Ocenaudio

1.8. 图片

  • Jietu: 截图工具
  • ImageOptim: 压缩图片

2. 付费互联网服务

  • Amazon prime
  • Curiositystream: 科普视频。 （已停用）
  • Leetcode 会员

2.1. 加密货币

  • Coin Tick: 在系统通知栏看价格。

3. 硬件

  • 电脑： Macbook Pro x3, Mac Studio x1
  • 显示器：Dell 2722QC
  • 鼠标：Logitech G502 Hero
  • 键盘：IKBC C87, HHKB
  • 耳机：Bose QC35
  • NAS: 群晖
